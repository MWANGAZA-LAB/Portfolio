## ğŸš€ Getting Started

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/MWANGAZA-LAB/Portfolio.git
   cd Portfolio
   ```

2. **Start local server**
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Or simply open index.html in your browser
   ```

3. **View in browser**
   ```
   http://localhost:8000
   ```
